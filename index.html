📰 AI 뉴스 자동화 파이프라인

RSS로 최신 AI 뉴스를 수집하고, AI로 한국어 브리핑을 자동 생성합니다.

동작 흐름

1. 수집 — TechCrunch·The Verge·VentureBeat 등 RSS 피드에서 최신 기사를 모읍니다.
2. 요약 — OpenAI API(gpt-4o)로 한국어 일일 뉴스 브리핑을 생성합니다.
3. 저장 — 날짜별 Markdown 보고서로 output/ 폴더에 기록합니다.

수집 소스

  • 🔹 TechCrunch — Artificial Intelligence
  • 🔹 The Verge
  • 🔹 VentureBeat — AI

주요 설정

항목값
피드당 최대 기사 수5개
요약 모델gpt-4o
출력 형식Markdown 보고서
필수 환경변수OPENAI_API_KEY

실행 방법

pip install -r requirements.txt
python main.py
